Primer

we won't start with "hello world" here – as this would be too trivial...right?
instead we start, more advanced, by counting from 1 to 20:

<?xq 1 to 20?>

added anywhere inside your HTML file, will be rendered as:

What happened

What the web server did, was replace the part between the <?xq … ?> with the evaluated result of that XQuery.

Why XQuery

This already saved us a lot of typing, and counting, but XQuery can be far more useful

  • XQuery is a database query language, similar to SQL in some ways – and luckily very different in other ways.
  • XQuery actually knows what a tag is, thus it never produces invalid (X)HTML
  • XQuery is a full fledged programming language
  • XQuery has strong full-text capabilities
